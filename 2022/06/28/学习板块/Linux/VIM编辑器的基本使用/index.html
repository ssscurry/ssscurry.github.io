VIM编辑器的基本使用

#安装

sudo apt-get install vim

#创建

vim test.text
然后就会跳转到如下界面

按“a”即可输入，左下角会显示“插入”

#保存

这个时候按ctrl+S并非保存，而是暂停终端，但此时输入都是会被保留的，在按ctrl+Q之后，这期间输入的内容都会被显示出来，终端也恢复正常。

那么如何保存呢？
1、按esc
2、（注意要切成英文输入）输入 ：
3、之后跟不同的指令，常用的有

x 保存当前文档并且退出
q 退出
w 保存文档
q！ 退出VIM，不保存文档

保存完后就退回到最开始的界面

主文件夹出现保存的文件

#如何搜索

在最外层的时候按“/”

把要搜的内容输进去就行

注：如何再次打开之前的文件，继续使用vim进行编辑
直接“vim ‘文件名’”即可
